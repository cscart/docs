************************************
Как настроить модуль "Конструктор прайс-листов"
************************************

#. Перейдите в панель администратора и выберите вкладку **Модули > Управление модулями**.

#. Убедитесь, что модуль **"Конструктор прайс-листов"** *активирован*.

#. Нажмите на название модуля.

#. В поле под названием **"пароль для cron"** введите свой пароль для планировщика cron.

#. Нажмите кнопку **Сохранить**.
