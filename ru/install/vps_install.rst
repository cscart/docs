*********************
Настройка сервера VPS
*********************

.. warning::

    Предупреждение! Настройку сервера для живого интернет-магазина необходимо доверить профессионалам. 

    Данная инструкция может быть использована для тестовых интернет-магазинов.


Хостинг провайдеры предоставляют VPS серверы с чистой операционной системой, без дополнительного ПО. Для работы любого сайта или CMS на VPS сервере потребуется устанавливать веб-сервер и другое ПО.

Для работы CS-Cart подходит два вида веб-серверов:

*   Apache (легкая настройка)

*   Nginx (сложная настройка, быстрая работа сайтов)

Рекомендуем начать с установки веб-сервера Apache, для ознакомления с принципами настройки VPS. В случае услеха, вы сможете переустановить систему и настроить веб-сервер Nginx.
